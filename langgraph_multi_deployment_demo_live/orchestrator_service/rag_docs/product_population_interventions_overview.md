# Population Interventions (Product Overview)

## What it is
Population Interventions helps teams identify cohorts (registries), stratify risk, and manage intervention workflows.

## Core entities
- Registry: A saved cohort definition (inclusion/exclusion + filters).
- Measure: A clinical or operational metric (numerator/denominator).
- Intervention: An action plan applied to a subset of the registry (outreach, reminders, tasks).
- Assignment: Work items for care teams.

## Typical workflow
1. Define a registry (cohort).
2. Validate cohort size and characteristics.
3. Stratify or segment (risk bands).
4. Configure intervention criteria (who gets outreach).
5. Execute intervention (generate tasks, outreach list).
6. Monitor outcomes and iterate.

## Common KPIs
- Registry size and trend
- Outreach completion rate
- Conversion (e.g., visit scheduled)
- Measure improvement (pre/post)
